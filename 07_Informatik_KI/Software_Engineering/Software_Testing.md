# Software Testing

**Kurzdefinition:**
Systematische Prüfung von Software, um Fehler zu finden und sicherzustellen, dass sie den Anforderungen entspricht (Unit Tests, Integration Tests, System Tests).

**Kategorie:**
Informatik & KI → Software Engineering

**Relevanz:**
Fehler im Live-Betrieb sind teuer und imageschädigend; Testautomatisierung ist daher essentiell.同期
