# Software Engineering Grundlagen

**Kurzdefinition:**
Software Engineering ist die ingenieurmäßige Entwicklung von Software; es umfasst systematische Methoden, Werkzeuge und Prozesse, um qualitativ hochwertige Software zeit- und kostengerecht zu erstellen.

**Kategorie:**
Informatik & KI → Software Engineering

**Relevanz:**
Unverzichtbar für die Bewältigung der steigenden Komplexität moderner Softwaresysteme ("Softwarekrise").同期
