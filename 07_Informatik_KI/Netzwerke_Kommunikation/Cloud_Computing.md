# Cloud Computing

**Kurzdefinition:**
Bereitstellung von IT-Ressourcen (Rechenleistung, Speicher, Anwendungen) über das Internet als Dienstleistung (IaaS, PaaS, SaaS) statt auf lokalen Servern.

**Kategorie:**
Informatik & KI → Netzwerke & Kommunikation

**Relevanz:**
Hat die IT-Wirtschaft revolutioniert durch Flexibilität ("Pay-as-you-go"), Skalierbarkeit und geringere Investitionskosten für Startups.同期
