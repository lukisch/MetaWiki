# Stochastische Integration

**Kurzdefinition:**
Die stochastische Integration (insbesondere das Ito-Integral) ist ein mathematisches Konzept zur Integration bezüglich eines stochastischen Prozesses, wie zum Beispiel der Brownschen Bewegung. Da diese Prozesse nirgends differenzierbar sind, erfordern sie eine spezielle Integrationslehre.

**Kategorie:**
Mathematik → Finanzmathematik

**Relevanz:**
Sie ist das unverzichtbare mathematische Fundament für die moderne Finanztheorie und wird genutzt, um die Dynamik von Aktienkursen und Zinsen in Echtzeit-Modellen präzise zu beschreiben.
