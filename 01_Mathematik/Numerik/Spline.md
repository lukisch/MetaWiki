# Spline

**Kurzdefinition:**
Ein Spline ist eine glatte, stückweise zusammengesetzte Polynomfunktion zur Interpolation von Datenpunkten. An den Übergangsstellen sind die Teilstücke so konstruiert, dass die Gesamtkurve ansprechend glatt (stetig differenzierbar) ist.

**Kategorie:**
Mathematik → Numerische Mathematik

**Relevanz:**
Splines werden intensiv im computergestützten Design (CAD) zur Modellierung geschwungener Formen sowie in der Statistik zur flexiblen Glättung verrauschter Datenreihen eingesetzt.
