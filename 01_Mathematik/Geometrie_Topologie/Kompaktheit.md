# Kompaktheit

**Kurzdefinition:**
Kompaktheit ist eine topologische Eigenschaft, die verallgemeinert, dass eine Menge abgeschlossen und beschränkt ist. Formal wird sie meist dadurch definiert, dass jede offene Überdeckung der Menge eine endliche Teilüberdeckung besitzt.

**Kategorie:**
Mathematik → Geometrie & Topologie

**Relevanz:**
Kompaktheit garantiert oft die Existenz von Lösungen für Extremwertprobleme und stellt sicher, dass stetige Funktionen auf diesen Mengen wohldefiniertes Verhalten zeigen.
