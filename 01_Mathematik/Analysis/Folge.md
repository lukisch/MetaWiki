# Folge

**Kurzdefinition:**
Eine Folge ist eine geordnete Liste von mathematischen Objekten, bei der jedem Index aus einer Indexmenge (meist die natürlichen Zahlen) genau ein Element zugeordnet ist. Folgen können endlich oder unendlich sein und auf Konvergenz gegen einen Grenzwert untersucht werden.

**Kategorie:**
Mathematik → Analysis

**Relevanz:**
Folgen bilden die diskrete Basis für konvergente Prozesse und sind entscheidend für die Definition von Reihen sowie für iterative Approximationsverfahren in der numerischen Mathematik.
