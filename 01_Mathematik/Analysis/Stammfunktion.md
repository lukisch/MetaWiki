# Stammfunktion

**Kurzdefinition:**
Eine Stammfunktion einer gegebenen Funktion ist eine Funktion, deren Ableitung wieder die ursprüngliche Funktion ergibt. Der Prozess der Bestimmung einer Stammfunktion wird als Aufleitung oder Integration bezeichnet.

**Kategorie:**
Mathematik → Analysis

**Relevanz:**
Nach dem Hauptsatz der Differential- und Integralrechnung ist die Kenntnis der Stammfunktion der Schlüssel zur einfachen Berechnung bestimmter Integrale und damit zur Bestimmung von Flächeninhalten unter Kurven.
