# Differentialgleichung

**Kurzdefinition:**
Eine Differentialgleichung ist eine Gleichung, die eine unbekannte Funktion und deren Ableitungen miteinander in Beziehung setzt. Sie beschreibt, wie sich eine Größe in Abhängigkeit von ihrem aktuellen Zustand oder anderen Parametern zeitlich oder räumlich verändert.

**Kategorie:**
Mathematik → Analysis

**Relevanz:**
Sie sind das primäre Werkzeug zur mathematischen Modellierung dynamischer Systeme in der Physik, Biologie, Chemie und den Ingenieurwissenschaften, um Vorhersagen über die Entwicklung komplexer Prozesse zu treffen.
