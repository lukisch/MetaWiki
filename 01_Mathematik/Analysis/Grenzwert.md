# Grenzwert

**Kurzdefinition:**
Ein Grenzwert (Limes) beschreibt den Wert, dem sich eine Funktion oder eine Folge unendlich nähert, wenn die unabhängige Variable oder der Index gegen einen bestimmten Wert oder gegen Unendlich strebt. Er bildet das theoretische Fundament für die gesamte Infinitesimalrechnung.

**Kategorie:**
Mathematik → Analysis

**Relevanz:**
Der Grenzwert ermöglicht die präzise Definition von Grundkonzepten wie Stetigkeit, Differenzierbarkeit und Integration durch den Umgang mit unendlich kleinen oder großen Größen.
