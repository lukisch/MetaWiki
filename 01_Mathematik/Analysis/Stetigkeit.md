# Stetigkeit

**Kurzdefinition:**
Eine Funktion gilt als stetig, wenn kleine Änderungen des Eingangswerts lediglich kleine Änderungen des Funktionswerts zur Folge haben, was anschaulich bedeutet, dass ihr Graph keine Sprünge oder Unterbrechungen aufweist. Formal wird dies über das Epsilon-Delta-Kriterium oder Grenzwerte definiert.

**Kategorie:**
Mathematik → Analysis

**Relevanz:**
Stetigkeit ist eine notwendige Voraussetzung für viele fundamentale Sätze der Analysis, wie den Zwischenwertsatz oder die Existenz von Extrema auf kompakten Mengen.
