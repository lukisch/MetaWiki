# Partielle Ordnung

**Kurzdefinition:**
Eine partielle Ordnung ist eine binäre Relation auf einer Menge, die reflexiv, antisymmetrisch und transitiv ist. Sie erlaubt den Vergleich einiger, aber nicht notwendigerweise aller Elemente der Menge (im Gegensatz zur Totalordnung).

**Kategorie:**
Mathematik → Diskrete Mathematik

**Relevanz:**
Sie findet breite Anwendung in der Informatik zur Beschreibung von Abhängigkeiten in Prozessen, Hierarchien in Datenstrukturen und zur Analyse von Kausalität in verteilten Systemen.
