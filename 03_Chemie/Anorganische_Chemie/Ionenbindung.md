# Ionenbindung

**Kurzdefinition:**
Die Ionenbindung ist eine chemische Bindungsart, die auf der elektrostatischen Anziehung zwischen entgegengesetzt geladenen Ionen (Kationen und Anionen) basiert. Sie entsteht meist durch den Übergang von Elektronen von Metall- auf Nichtmetallatome.

**Kategorie:**
Chemie → Anorganische Chemie

**Relevanz:**
Sie ist verantwortlich für die Bildung von Salzkristallen mit hohen Schmelzpunkten und spielt eine entscheidende Rolle für die Leitfähigkeit von Elektrolyten.同期
