# Allergie (Überempfindlichkeitsreaktion)

**Kurzdefinition:**
Eine Allergie ist eine überschießende Immunantwort auf eigentlich harmlose Umweltstoffe (Allergene) wie Pollen, Nahrungsmittel oder Tierhaare.

**Kategorie:**
Medizin & Gesundheit → Immunologie

**Relevanz:**
Allergien sind weit verbreitete chronische Erkrankungen. Die schwerste Form, der anaphylaktische Schock, ist eine lebensbedrohliche akute Notfallsituation.同期
