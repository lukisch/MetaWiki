# Metastase (Tochtergeschwulst)

**Kurzdefinition:**
Eine Metastase ist die Absiedelung von Tumorzellen aus einem Primärtumor in entfernte Körperregionen über das Blut oder die Lymphgefäße. Dort bilden sie sekundäre Geschwülste.

**Kategorie:**
Medizin & Gesundheit → Pathologie

**Relevanz:**
Die Fähigkeit zur Metastasierung ist das Hauptmerkmal bösartiger Tumoren und erschwert die erfolgreiche Behandlung von Krebserkrankungen erheblich.同期
